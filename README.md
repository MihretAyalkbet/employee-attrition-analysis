# Employee Attrition Analysis

This project analyzes whether overtime is associated with employee attrition and highlights retention risk indicators using statistical analysis.

## Dataset

This project uses the IBM HR Analytics Employee Attrition dataset.

Due to sharing considerations, the dataset is not included in this repository.  
To run the notebook, please download the dataset and place it in a `data/` folder as:

- `WA_Fn-UseC_-HR-Employee-Attrition.csv`

## Files
- `Burnout_retention_analysis.ipynb` – full notebook analysis
- `Burnout_retention_executive_summary.pdf` – one-page executive summary

## Methods
- Descriptive statistics
- Correlation analysis
- Chi-square test
- Data visualization

## Key finding
Employees working overtime showed substantially higher attrition than employees who did not.

## Note
This analysis shows association, not causation.
